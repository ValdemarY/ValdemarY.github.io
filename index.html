Политика конфиденциальности приложения Weather Forecast

Я (ValdemarY) как разработчик приложения Weather Forecast со всей серьезностью отношусь к Вашей конфиденциальности.

Ваши данные, которые вы передаете при входе в приложение, не передаются третьим лицам и надежно хранятся в локальном хранилище приложения.

Далее я привожу список данных, которые приложение использует для своей работы

  • Access_Token - При входе в приложение генерируется токен, с помощью которого приложение может получить доступ к вашему имени аккаунта.
  • name - Weather Forecast использует имя вашего профиля для отображения его в SideBar'e приложения.
